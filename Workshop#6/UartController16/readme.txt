How to use UartController16 on linux:

1. Install ncurses library.
2. Download this directory(UartController16).
3. Find a folder using the terminal and go to it(cd UartController16/).
4. Run ʼmakeʼ.
5. You should have the program compile automatically.
6. Run ./UartController16 -d tty  [-b baudrate] .
   (for example ./UartController16 -d /dev/ttyUSB0 115200)
7. Use text prompts to control the LEDs, frequency and duty cycle.
8. Rejoice!
