How to use UartController on linux:

1. Install ncurses library.
2. Download this directory(UartController).
3. Find a folder using the terminal and go to it(cd UartController/).
4. Run ʼmakeʼ.
5. You should have the program compile automatically.
6. Run ./UartController -d tty  [-b baudrate] .
   (for example ./UartController -d /dev/ttyUSB0 115200)
7. Use text prompts to control the LEDs.
8. Rejoice!
